nX-U16 Compiler (Version 1.0 BETA)

-# Do là nX-U16 nên chỉ hỗ trợ dòng máy fx-580 VNX.

Lưu ý :

  • Đọc cú pháp trong Instruction.txt hoặc Instruction_VN.txt
  • Đây mới chỉ là bản BETA nên vẫn có thể xuất hiện lỗi ## Cách sử dụng (Đối với Android):
  • Tải cái này về rồi giải nén
  • Tải TermuX về, rồi cấp quyền truy cập mọi file bằng termux-setup-storage
  • Đưa con trỏ về thư mục có chứa Compiler: cd <đường dẫn đến thư mục có nX-U16_Compiler>/nX-U16_Compiler
  • Tải package Python: pkg install python
  • Chạy compile : python launch.py ## Đối với ai muốn dùng lệnh để inject thẳng vào hc-inj.txt trong giả lập máy tính của Telecom :
  • Tùy vào dòng máy, mở "Tùy chọn nhà phát triển" rồi mở "Gỡ lỗi qua Wifi" lên (có thể mở thêm gia hạn ADB gì đó)
  • Mở "Ghép nối với thiết bị qua Wifi", lấy cái Local IP và mã
  • Vào TermuX, nhập adb pair <Local IP> rồi Enter. Sau đó nhập mã mình vừa lấy vào
  • Cuối cùng vào lại "Gỡ lỗi qua Wifi", lấy IP từ "Địa chỉ IP và cổng". Mở TermuX và gõ adb connect <IP>
  • Lúc này, bạn đã có thể dùng lệnh *export_data để xuất output đúng form vào hc-inj.txt
