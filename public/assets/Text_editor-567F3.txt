### Cách sử dụng:
Tiếp theo bấm [0] và [Ac] rồi sử dụng như Text editor plus
[▲]: Tăng giá trị
[▼]: Giảm giá trị
[▶]: Tăng vị trí kí tự
[◀]: Giảm vị trí kí tự
[1]: Set giá trị thành "A"
[2]: Set giá trị thành "a"
[3]: Set giá trị thành F4
[4]: Set giá trị thành dấu cách 20
[5]: Set giá trị thành dấu cách 01
[6]: Cuộn xuống (Nhấn giữ để cuộn)
[7]: Cuộn lên (Nhấn giữ để cuộn)
[8]: Invert(Bôi đen) kí tự
[9]: Upsidedown("Lộn ngược") kí tự
[0]: Xóa invert và upsidedown
[Del]: -1 vị trí kí tự(Chức năng giống như nút [Del] bình thường)
[Ac]: Clear màn, chuyển về vị trí đầu tiên,xóa invert và upsidedown

Warning⚠️⚠️⚠️:
[=]: Để giả tắt máy ☠️☠️☠️
Cách hóa giải thì bấm [0],[9] hoặc [Ac]😏
